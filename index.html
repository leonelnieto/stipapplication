UDOT MoPEED Documentation

More on Projets to Expand Economic Development is a web pages to give details on UDOT projects to the State Transportation Investment Planning Workshop. MoPEED queries dataset a6xh-u32h hosted in UDOT's Open Data Catalog. The data is queried with an API and pulled as JSON, filtered and displayed with HTML, Javascript and JQuery. The primary filters are Workshop Category (workshop_cat) and PIN Status (pin_st_nm). Also for unfunded projects the commission approved projects column is used. Lastly, projects in design are projects with PIN Status scoping, Projects funded are projects with PIN Status Advertised.
